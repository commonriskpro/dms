## Use for DMS

Reference these assets when implementing:

- **Shell:** Topbar search, sidebar (e.g. Home, Dashboard, Customer, Deal, etc.), header actions (Check-In, Start Deal, notifications).
- **Homepage / Sales Dashboard:** Widgets for Appointments, Tasks, New Prospects, Sales Activity, Funnel, Email/SMS.
- **Customer detail:** Lead-first tab, road-to-sale bar, action strip (Phone, SMS, Email, Schedule Appt, Add Task, Disposition), Buyer/Co-Buyer/Referrer, Activity filters, Interested Vehicle.
